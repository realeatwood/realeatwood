dart学习笔记（1）

首先，
dartlang – 官方文档，是所有 dart 项目都会引入的基础库，不包含三方库。
cnDartlang – 中文的 dart 学习社区， 的一系列文章。

dart 中万物都是对象.
基本类型:

  • String,
  • Number:
    • int
    • double
  • Collections:
    • List
    • Set
    • Map
  • Booleans。
  • Runes > UTF-32编码字符串 ，dart的String是UTF-16 ,可以生成一些小图标、emoji表情
  • Symbols
    他们都继承自Object,默认初始值都是null。So ，就像 js 一样，可以调用很多方法～

同样包含有 var，const，final，声明变量。
var 可以声明任意类型的变量，忽略显式的声明。var String i = 10.33333333; // error
final 和 const 都是定义的常量，在声明的时候就必须初始化.区别是：const 必须用常量来初始化。

一个简单的 web，可能需要你安装 dart 编辑器，运行 Dartium（chorme专用dart浏览器） 来访问 html，html 需要这样引入 dart<script type="application/dart" src="/main.dart"></script>.
或者，你还可以采用dart2js命令行工具，将dart编译成为js,dart SDK 1.24以上提供。
