dart学习笔记（1）

首先，官方的：dartlang
dart中万物都是对象.
基本类型:String, Number, List, Maps, Booleans。他们都继承自Object,默认初始值都是null。So ，就像js一样，可以调用很多方法～

同样包含有var，const，final，声明变量。
var可以声明任意类型的变量，忽略显式的声明。var String i = 10.33333333; // error
final和const都是定义的常量，在声明的时候就必须初始化.区别是：const必须用常量来初始化。
